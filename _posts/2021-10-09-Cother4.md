# Tableau?

## 핵심 개념
- 차원과 측정값
- 연속형과 불연속형
- Level of Detail
- 네 가지 계산
- 시트, 대시보드, 스토리

## 필수 기능
- 데이터 준비
- 필터
- 이중축 + Combined Axis Chart
- 분석 패널
- 지도

## Level UP
- 계산된 필드 + IF 함수
- 매개변수
- 집합
- 테이블 계산의 작동 원리
- INDEX, SIZE, RANK, TOTAl
- WINDOW
- 중첩된 테이블 계산
- 테이블 계산과 필터 활용
- LOD 표현식
  - VLOD
  - INCLUDE LOD
  - EXCLUDE
  - FIXED
  - LOD 표현식과 필터 활용
