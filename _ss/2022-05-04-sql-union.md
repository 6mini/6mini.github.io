## UNION
- 컬럼명이 같아야 한다.(같지 않을 경우 AS를 사용하여 같게 만들어주면 된다.)
- 컬럼별 데이터타입이 같아야 한다.
- 두 테이블에 중복으로 들어있는 데이터는 DISTINCT 된다.

```sql
SELECT *
FROM User_recent

UNION

SELECT *
FROM User_past;
```

## UNION ALL
- 중복되는 데이터들을 생략하지 않고 원본의 모든 데이터를 단순히 이어붙인 결과

```sql
SELECT *
FROM User_recent

UNION ALL

SELECT *
FROM Users_past;
```
